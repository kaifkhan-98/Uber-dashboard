# Uber-dashboard
1. Project Title / Headline
Uber Ride Booking & Operations Analytics Dashboard | Power BI

2. Short Description
Designed an interactive Power BI dashboard to analyze Uber ride bookings, revenue, cancellations, vehicle performance, and customer behavior. The dashboard helps identify booking trends, operational efficiency, cancellation patterns, and revenue-driving factors through dynamic visualizations and KPI monitoring.

3. Tech Stack
Power BI
Power Query
DAX (Data Analysis Expressions)
Data Modeling
Microsoft Excel / CSV
Data Visualization

4. Data Source
Uber Ride Booking Dataset

Booking ID
Date & Time
Booking Status
Customer ID
Vehicle Type
Pickup & Drop Location
Booking Value
Customer Cancellations
Driver Cancellations
Incomplete Rides
Ratings
